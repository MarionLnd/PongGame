Pong Game


Bienvenue sur le jeu PongGame ! Il s'agit d'un jeu de Ping Pong sur navigateur.
Avant de lancer une partie, vous devez selectionner si vous voulez jouer avec la souris, ou le clavier. Si vous perdez la balle, appuyez sur la touche espace du clavier pour relancer la balle.
